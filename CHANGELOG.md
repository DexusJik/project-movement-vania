# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - The Prototype (JS/Matter.js)
- Initial implementation of the movement controller.
- Basic morphing (Standing $\leftrightarrow$ Superman $\leftrightarrow$ Ball).
- Implementation of the "Robustness Pass" (probe-based grounding).
- Initial Glide physics logic.

## [0.2.0] - The Pivot (Godot Migration)
- **Decision**: Pivot to Godot 4.x for improved physics stability and toolset.
- **Action**: Archive JS prototype and initialize Godot project structure.
- **Update**: Established "Dev Hub" repository structure for design documentation.
